# News RSS Reader

## Requirements
- C++ compiler
- nCurses

## Getting Started

1. Clone the repository
```bash
git clone https://github.com/ignabelitzky/news_rss_reader.git
cd news_rss_reader
```
2. Build the project using the Makefile
```bash
make
```
3. Run the executable
```bash
./main
```
